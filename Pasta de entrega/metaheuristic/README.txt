Para executar o arquivo é necessário possuir Python 3 e executar em um prompt de comando "py ballsAndBins.py"
Durante a execução, é pedido o nome do arquivo de entrada, que deve ser passado a partir da pasta em que está o código da metaheuristica. O arquivo deve ser inserido com a extensão (.txt).
Após o nome do arquivo, o programa pede, nessa ordem:
- O tamanho da fila (parâmetro do Late Acceptance Hill Climbing) (valor inteiro).
- A quantidade máxima de interações permitidas sem melhora para que o programa pare (valor inteiro)
- O tempo máximo de execução (valor inteiro)
